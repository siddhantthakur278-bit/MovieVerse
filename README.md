# MovieVerse
A sleek, responsive dashboard where users can discover trending movies, search for specific titles, and filter them by genre or rating.

# MovieVerse Explorer
## Purpose
A dynamic web application to browse movies using the TMDB API. Built to demonstrate proficiency in asynchronous JS and Array Higher-Order Functions.
## Features
* Real-time search by movie title.
* Filter by Genre (Action, Comedy, etc.).
* Sort by Release Date and User Rating (Array.sort).
* Responsive Grid Layout.
## Technologies
* HTML5, CSS3 (Flexbox/Grid), JavaScript (ES6+), TMDB API.
